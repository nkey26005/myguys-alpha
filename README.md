# MyGuys Messenger Alpha 1.1

Настоящий реал-тайм мессенджер.

### Что нового в 1.1:
- Постоянное хранение сообщений (SQLite)
- Время отправки сообщений
- Индикатор "печатает..."
- Автоскролл
- Улучшенный интерфейс
- Подготовка к деплою

### Как запустить:
```bash
git pull origin main
npm install
npm start
```

Открывай: http://localhost:3000
