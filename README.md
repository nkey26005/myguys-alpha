# MyGuys Messenger

**Alpha 1.0** — твой настоящий мессенджер.

Простой, быстрый и красивый real-time чат на Socket.io.

### 🚀 Как запустить локально:

```bash
npm install
npm run dev
```

Затем открой в браузере: **http://localhost:3000**

Можешь открыть несколько вкладок или окон — сообщения будут приходить мгновенно.

### Особенности Alpha 1.0:
- Реальное время (WebSockets)
- Красивый тёмный дизайн
- Общий чат для всех
- Сохранение последних 50 сообщений
- Простая авторизация по имени

Следующие версии: личные сообщения, комнаты, авторизация, эмодзи, файлы и т.д.
