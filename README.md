# MyGuys Messenger

**Alpha 1.1**

Простой, но уже настоящий реал-тайм мессенджер.

### Что нового в 1.1:
- Сообщения сохраняются (пока сервер работает)
- Показывается время отправки
- Улучшенный дизайн
- Более стабильная работа

### Как запустить:

```bash
npm install
npm start
```

Открыть: http://localhost:3000
